# Two-Sample Binomial Log-Likelihood

Computes the log-likelihood for two independent binomial samples.

## Usage

``` r
two_sample_binomial_loglik(x0, x1, n0, n1, p0, p1)
```

## Arguments

- x0:

  Number of events in group 0

- x1:

  Number of events in group 1

- n0:

  Sample size of group 0

- n1:

  Sample size of group 1

- p0:

  Probability in group 0

- p1:

  Probability in group 1

## Value

Log-likelihood value
