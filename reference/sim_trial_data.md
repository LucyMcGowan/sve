# Simulated Vaccine Trial Data

A simulated data set.

## Usage

``` r
sim_trial_data
```

## Format

### `sim_trial_data`

A data frame with 500 rows and 60 columns:

- id:

  ID

- vaccination:

  Vaccine status (1 = vaccinated, 0 = unvaccinated)

- age:

  Age

- baseline_risk:

  Baseline risk (low, medium, or high)

- time:

  Survival time

- status:

  Censoring status
